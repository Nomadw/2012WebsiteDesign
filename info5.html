The TCP/IP Protocol and layer protocols

The TCP/IP model, sometimes known as the internet model, is a framework developed in the 1970s by DARPA, part of the United States defense department, , to descrive the framework of all protocols to allow computers to communicate over a network.


Protocol Description
File Transfer Protocol (FTP) Used to transfer files over a TCP-based network, such as the internet.
Secure Mail Transfer Protocol (SMTP) Internet standard for transfering E-Mail across IP networks.
Hypertext Transfer Protocol (HTTP) Networking protocol for connected hypermedia data systems. The foundations of data communcation throughout the World Wide Web.
Hypertext Transfer Protocol Secure (HTTPS) Combination of HTTP and SSL/TLS to provide encrypted communication.
Dynamic Name System (DNS) Provides a gateway to a dynamically changing IP.
